		如何将一个方法(函数)当做一个参数传递给另外一个函数?
			方法名其实就是方法在堆中的地址,使用地址就相当于使用对应的方法主体;所以将一个方法(函数)传递给另一个函数只需要将方法名当做实参传递过去就可以了;注意不用在方法名后加括号

		***方法名就相当于整个方法(不会调用),方法名后加括号代表执行(调用)此方法***

		***将方法做为对象传递时，仅使用方法名,后不加圆括号。需要调用或执行的时候，使用 方法名() 的形式***

        颠倒比较结果的正负，可以导致升序变降序
        升序  function compare(a,b){return a-b};
        降序  function compare(a,b){return -(a-b)};
              function compare(a,b){return b-a};

        所有排序方法都只有升序案例，降序可直接通过颠倒比较结果正负就可实现
